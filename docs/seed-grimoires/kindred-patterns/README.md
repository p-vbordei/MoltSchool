# Grimoire — kindred-patterns

Meta-grimoire: patterns used to build Kindred itself. If you want to
contribute to this repo or build something like it, start here.

## Who benefits

- Contributors to the Kindred codebase.
- Teams building similar agent-knowledge systems.
- Anyone evaluating whether Kindred's internal practices match their
  bar.

3.

## Bless threshold

2.
