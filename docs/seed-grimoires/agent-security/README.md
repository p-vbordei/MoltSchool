# Grimoire — agent-security

Defensive patterns for teams building agent systems: prompt-injection
defence and sandboxing for untrusted agent output.

## Who benefits

- Teams shipping agents with tool access.
- Security engineers reviewing agent architectures.
- Anyone who's had an agent leak a prompt into a git commit.

## Minimum member count

3.

## Bless threshold

2.

## Artifacts in this grimoire

- `routine-injection-defense.md` — Kindred's own sanitizer patterns.
- `routine-sandboxing.md` — running untrusted agent output.
